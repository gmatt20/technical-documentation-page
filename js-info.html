JavaScript

1. Numbers and Math

Basic arithmetic operations and using mathematical functions.

let x = 10 + 5; // 15 let X-UA-Compatible = 10 - 5; // 5 let x = 10 * 5; // 50 let x = 10 / 5; // 2 let x = Math.sqrt(16); // 4

2. Strings

Creating and manipulating strings using various methods.

let greeting = "Hello, " + "World!"; // "Hello, World!" let upper = "hello".toUpperCase(); // "HELLO" let lower = "hello".toLowerCase(); // "hello" let template = `Hello, ${name}!`; // Template Literal

3. console.log

Outputting information to the browser console for debugging.

console.log("Hello, World!");

4. Variables

Declaring variables using var, let, and const.

var x = 10; let y = 20; const z = 30;

5. Booleans and If-Statements

Using boolean values and conditional statements.

let isTrue = true; if (isTrue) { console.log("It's true!"); } else { console.log("It's false!"); }

6. Functions

Defining and calling functions.

function greet(name) { return `Hello, ${name}!`; } console.log(greet("Alice"));

7. Objects

Creating and manipulating objects.

let person = { name: "John", age: 30, greet: function() { console.log(`Hello, my name is ${this.name}`); } }; person.greet();

8. Document Object Model (DOM)

Selecting and manipulating DOM elements.


Selecting an element

const element = document.querySelector('.my-element');

Selecting an ID

const element = document.getElementById("id");

innerHTML

const element = document.querySelector('.class').innerHTML;

Changing content

const element = document.querySelector('.class').innerHTML = "new content";

Adding a class

element.classList.add('new-class');

Event Handling

element.addEventListener('click', function() { console.log('Element clicked!'); });

JS has a unique feature that can react to events. Like when a user clicks on an HTML element, JS will execute some code. onclick = "" is the most common example of this. Here are more:


onmouseover = "" onmouseout = "" onmousedown = "" onmouseup = "" onmousemove = "" onmousemove = "" onkeydown = "" onkeypress = ""

Rather than always having to rely on HTML attributes, JS can also do the same within the JS code. JS can do this by using the: .addEventListener("event", function, useCapture);


9. Loops

Using for, while, and do...while loops to iterate over data.

for (let i = 0; i < 5; i++) { console.log(i); }

10. Arrays

Creating and manipulating arrays using various methods.

let fruits = ["apple", "banana", "cherry"]; fruits.push("date"); console.log(fruits); // ["apple", "banana", "cherry", "date"]

11. JSON and Local Storage

Understand storing and retrieving data using localStorage. Save data to local storage by:

localStorage.setItem('key', value);

Read data from local storage:

let lastname = localStorage.getItem(key);
